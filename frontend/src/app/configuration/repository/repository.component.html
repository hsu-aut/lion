Graph Database Configuration

In order to build the ABox of your project, the LiOnS tool uses a GraphDB. Using LiOnS you can manage your repositories as well as the relevant TBoxes within those repositories. This enables you to effecitevely manage you projects.

Set Repositories: In case you already have created a repository, you can set it here is the current repository.

Define relevant TBoxes: You can set the relevant TBoxes for your modelling project. This will impact the size and modelling abilities of your ABox.

Managing repositories: You can create or delete repositories in order to manage you modelling projects.

Important notice

Please be aware, that clearing or removed any repoistory will lead to a loss of data.
