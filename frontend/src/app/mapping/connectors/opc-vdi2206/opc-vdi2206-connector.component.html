Engineering Artefact to RDF Mappings (Connections)

OPC UA and VDI 2206

By using the the mapping functionalities of LiOnS, one can create an ABox based on artefacts. If one wants to express relations between individuals that have been mapped from different artefacts, these relations have to be created manually.
With this Module, you can connect a system or module of VDI 2206 and an OPC UA Server. The connection specifies that the system or module contains an OPC UA Server.

Important notice

Please be aware that only individuals of VDI 2206 and OPC UA are shown. In order to connect different mapping results, refer to the corresponding connection tab in the sidebar.

  • Review Connections
  • Create Connection
